MEDIA PEMBELAJARAN INFORMATIKA KELAS VII

Hardware • Software • Brainware • Utility Software • Scratch • Sistem Operasi

Hardware

Hardware adalah perangkat keras komputer yang dapat dilihat dan disentuh secara fisik. Hardware berfungsi sebagai alat input, proses, penyimpanan dan output data.

Contoh Hardware

  • Keyboard
  • Mouse
  • Monitor
  • Printer
  • RAM
  • Motherboard
  • Harddisk

Software

Software adalah perangkat lunak berupa kumpulan instruksi yang digunakan untuk menjalankan komputer. Software tidak mempunyai bentuk fisik.

Contoh Software

  • Windows
  • Linux
  • Android
  • Microsoft Word
  • Google Chrome

Brainware

Brainware adalah manusia yang menggunakan atau mengelola sistem komputer.

  • User
  • Operator
  • Programmer
  • Administrator
  • Teknisi

Perbedaan Hardware, Software dan Brainware

Hardware Software Brainware
Perangkat Fisik Program Manusia
Dapat disentuh Tidak berwujud Pengguna sistem

Utility Software

Utility Software adalah software yang membantu pemeliharaan dan perawatan komputer.

  • Antivirus
  • Disk Cleanup
  • Backup Software
  • Defragmenter
  • Recovery Software

Scratch

Scratch adalah bahasa pemrograman visual berbasis blok yang cocok untuk pemula.

  • Motion
  • Looks
  • Sound
  • Events
  • Control
  • Sensing
  • Variables
  • Operators

Sistem Operasi

Sistem operasi adalah perangkat lunak yang mengelola seluruh sumber daya komputer.

  • Windows
  • Linux
  • Ubuntu
  • Debian
  • Android
  • iOS
  • macOS
