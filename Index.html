Hardware, Software, Brainware, Utility Software, Scratch dan Sistem Operasi

Hardware

Hardware adalah perangkat keras komputer yang dapat dilihat dan disentuh secara fisik. Hardware berfungsi untuk menerima input, memproses data, menyimpan data dan menampilkan hasil. Contohnya keyboard, mouse, monitor, printer, RAM, motherboard dan harddisk.

Jenis Hardware

  • Input Device
  • Output Device
  • Processing Device
  • Storage Device

Software

Software adalah perangkat lunak berupa kumpulan instruksi yang digunakan untuk menjalankan komputer. Software tidak dapat disentuh secara fisik. Contohnya Windows, Linux, Android, Microsoft Word dan Google Chrome.

Brainware

Brainware adalah manusia yang menggunakan atau mengelola sistem komputer. Tanpa brainware, hardware dan software tidak dapat dimanfaatkan secara maksimal.

  • User
  • Operator
  • Programmer
  • Administrator
  • Teknisi

Utility Software

Utility Software adalah software yang membantu pemeliharaan sistem komputer.

  • Antivirus
  • Disk Cleanup
  • Backup Software
  • Defragmenter
  • Compression Software
  • Recovery Software

Scratch

Scratch adalah bahasa pemrograman visual berbasis blok yang memudahkan pemula belajar coding.

  • Motion
  • Looks
  • Sound
  • Events
  • Control
  • Sensing
  • Variables
  • Operators

Sistem Operasi

Sistem Operasi adalah software yang mengatur seluruh sumber daya komputer.

  • Windows
  • Linux
  • Ubuntu
  • Debian
  • Android
  • iOS
  • macOS
